## ドメイン駆動設計における値オブジェクトとは

- システム固有の値を表現するもの
- **「オブジェクト自身が知っていること」** を値オブジェクトの実装とすることで、下記のようなメリットがある
    - (メソッドの引数や返り値などでの)表現力が増す
    - プログラム中に不正な値を存在させない
    - 誤った代入を防ぐ
    - ロジックの散在を防ぐ
